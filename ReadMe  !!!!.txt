A desktop application of library management created using java swings, the main object of this project is to use 
Junit Test and generate the report of test, also to use PMD which is a framework for analyzing Java source code 
this plugin enables to generate a report that describes the state of your source code after the analyizing 
has finished, and also to use sonarLint which does almost the same job as the PMD, the difference is that the PMD 
classify the code quality errors.


Note 1 : read the report of the project so you can easily understand how the application is being executed
Note 2 : you will find all the generated reports of these plugins in project folder and note that you have 
the right to do changes and regenerate them again and again. :)
